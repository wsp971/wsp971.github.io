最近的生活

发表于 2016-05-08   |   分类于 生活   |     |   阅读次数

这是离开微信朋友圈的第二周，一开始强制自己不去看微信，不去刷朋友圈，渐渐地也适应了，离开了朋友圈，生活似乎变得更加简单， 工作的时候似乎更加专注，一个人来来往往，只在工作的路上遐想未来的模样。
今天是母亲节，给家里打了电话，想问候一下，不料老妈还没下班，没接着。可能她都不记得这是一个母亲的节日了吧。前段时间有次老妈打来电话，正好刚睡着，听到电话的瞬间，非常生气，没说几句就气冲冲的挂断了。之后觉得做的不对，在第二天给母亲解释当时的情况，后来才得知，老妈当晚一晚上都没睡着，想了一晚上。突然觉得长大了，有时候自己不高兴了，真的不应该在父母面前像个孩子一样随便的发泄自己的情绪，父母不在身边，他们真的会多想。
上个月去了凤凰，给老妈早早就买好了母亲节礼物，可惜，妈妈现在的地方太远，取快递不方便，打电话说要寄，老妈却嫌麻烦，不让寄，让过年的时候带过来；不管怎么样，我真的是爱您的。愿您健康快乐。幸福长寿~

生活窘迫，来了就赏点吧~~
