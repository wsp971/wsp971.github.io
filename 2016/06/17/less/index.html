less使用

发表于 2016-06-17   |   分类于 技术   |     |   阅读次数

less是什么

less 是一种动态样式语言，属于CSS预处理语言的一种，它使用类似CSS的语法，为CSS赋予动态语言的特性。如：变量，继承，运算，函数等，更方便CSS的编写和维护。

less编译

  • 客户端编译工具：koala编译，下载地址： koala
  • nodejs编译

  • 浏览器端使用
    初次使用，先用客户端编译工具koala编译体验，项目中应该用基于node的项目打包工具编译会更方便。

    less语法

    注释

    跟js 一样，有两种注释，如下：

  • /**/ ：可以被编译在css文件中的注释。例如 

/*我是可以被编译进css的注释*/
  • // ：不可以编译进css文件中的注释。例如 
// 我是不可以被编译进css文件的注释。

变量

变量用@符号来声明，例如： @test-width:300px ,在样式中可以这样引用：.box{width:@test-width; height:@test-width}
执行编译后 会生成样式如下：

.box {
  width: 300px;
  height:300px;
}

混合（mixin）

  • 混合变量

例如：.border{border:solid 1px red;}

  • 带参数的混合, 例如：.border-radius(@radius){css代码}

    我们新建一个box2来混合box,并且新加样式：

    .box2{
    .box;
    color:red;
    background-color: #green;
}

    编译后：

    .box2 {
  width: 300px;
  height: 300px;
  color: red;
  background-color: #green;
}

    可以看到，box2继承了box的属性，less中叫混合。

  • 默认参数的混合：.border-radius(@radius:5px){css代码} 例如：

    .border_01(@border_width){
   border:solid blue @border_width;
}
.box3{
    .border_01();
    width: @test-width;
    height: @test-width;
}

    编译后：

    .box3 {
  border: solid #0000ff 5px;
  width: 300px;
  height: 300px;
}

匹配模式

匹配模式类似于编程语言里的if else,或者更像 switch case，下面我们写个例子。

首先，实现一个朝上三角的css样式如下：

.sanjiao{
    width:0;
    height: 0;
    overflow: hidden;
    border-width: 50px;
    border-color:  transparent transparent red transparent;
    //兼容 ie6写法，透明边要写上dashed
    border-style: dashed dashed solid dashed;
}

现在写一个通用的 三角，他的朝向可以变化，颜色可以变化，大小也可以变化的样式，这时候用匹配模式。

//朝上三角
.trangle(top,@w:5px,@c:red){
    border-width: @w;
    border-color: transparent transparent @c transparent;
    border-style: dashed dashed solid dashed;
}
//朝下三角
.trangle(bottom,@w:5px,@c:red){
    border-width: @w;
    border-color: @c  transparent transparent transparent;
    border-style: solid dashed dashed  dashed;
}
//朝左三角
.trangle(left,@w:5px,@c:red){
    border-width: @w;
    border-color: transparent transparent  transparent @c;
    border-style: dashed dashed  dashed solid;
}
//朝右三角
.trangle(right,@w:5px,@c:red){
    border-width: @w;
    border-color: transparent @c transparent transparent;
    border-style: dashed solid dashed  dashed;
}
//这部分是每个trangle 的公共部分，公共部分用@_声明第一个匹配值
.trangle(@_,@w:5px,@c:red){
    width: 0;
    height: 0;
    overflow: hidden;   
}

现在我们可以任意的自定义三角了：

//10px 颜色为#ccc的向上三角
.sanjiao1{
    .trangle(bottom,10px,#ccc);
}
//5px,颜色为红色的向左三角
.sanjiao2{
    .trangle(left);
}

编译后：

.sanjiao1 {
  border-width: 10px;
  border-color: #cccccc transparent transparent transparent;
  border-style: solid dashed dashed  dashed;
  width: 0;
  height: 0;
  overflow: hidden;
}
.sanjiao2 {
  border-width: 5px;
  border-color: transparent transparent transparent #ff0000;
  border-style: dashed dashed  dashed solid;
  width: 0;
  height: 0;
  overflow: hidden;
}

运算

less中的数字、颜色、变量都可以参与运算、例如 + - * / 都可以。

写个例子：

//声明变量
@test-len:200px;
//box3
.box3{
    width:@test-len - 20;
    height: @test-len + 30px;
    background-color: #ccc-10;
}

编译后：

.box3 {
  width: 180px;
  height: 230px;
  background-color: background-color: #c2c2c2;;
}

可以看到，box3的宽度值已经变成180px,高度变为 230px。注意在加减过程中，可以不带单位，带上也无妨，另外颜色也可以做运算，只不过，平常我们基本不用。

嵌套规则

  • 对伪类使用，如 hover 或focus

我们实现一个ul列表如下：

<ul class="list">
    <li><a href="">这是测试文字</a><span>2016-06-18</span></li>
    <li><a href="">这是测试文字</a><span>2016-06-18</span></li>
    <li><a href="">这是测试文字</a><span>2016-06-18</span></li>
</ul>

写一些样式在上面，对比传统我们 .list .list>li .list>li a 等等各自一长串的写法，我们用less嵌套来写：

.list{
    width: 600px;
    margin:30px auto;
    padding: 0;
    list-style: none;
    //嵌套定义
    li{
        height: 30px;
        line-height: 30px;
        background-color: pink;
        margin-bottom: 5px;
    }
    a{
        float:left;
        //& 代表上一层选择器
        &:hover{
            color:red;
        }
    }
    span{
        float:right;
    }
}

编译后：

.list {
  width: 600px;
  margin: 30px auto;
  padding: 0;
  list-style: none;
}
.list li {
  height: 30px;
  line-height: 30px;
  background-color: pink;
  margin-bottom: 5px;
}
.list a {
  float: left;
}
.list a:hover {
  color: red;
}
.list span {
  float: right;
}

可以看到，最后也会编译成传统的写法，但是less 写法更加具有可读性，维护起来也一目了然。

@arguments 变量

@arguments包含了所有传递近来的参数，类似于 js 函数中的auguments变量。

写个简单例子：

.border-arg(@w:30px,@c:#ccc,@xx:solid){
    //用arguments代替写各个参数
    border:@arguments;
}
.border-test{
    .border-arg();
    width:300px;
    height:300px;
}

编译后：

.border-test {
  border: 30px #cccccc solid;
  width: 300px;
  height: 300px;
}

可以看到，用@arguments 比之前只是少些了一点点代码，了解即可。

避免编译

有时候我们需要输出一些不正确的css语法或者使用一些less不认识的专用语法，要输出这样的值，我们可以在字符串前面加上 ~ 例如：width:~' calc(100% - 35)'

简单例子：

.test_03{
    width:~'calc(300px-30px)';
}

编译后：

.test_03 {
  width: calc(300px-30px);
}

可以看到，加上~后的字符串，less编译过程中，并没有进行计算，而将其原封不动的复制了过去。
