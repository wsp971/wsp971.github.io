什么时候用到了节流函数？

有这样一个需求啊，就是当浏览器改变大小的时候，需要改改页面上的元素大小、这个你怎么做？很自然的想到了监听浏览器窗口的 resize 事件、于是我们写下以下代码：

var n=0;   
function resizeHandle(){
    console.log(new Date().getTime());
    console.log(++n);
}
window.onresize=resizeHandle;
