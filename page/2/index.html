less是什么

less 是一种动态样式语言，属于CSS预处理语言的一种，它使用类似CSS的语法，为CSS赋予动态语言的特性。如：变量，继承，运算，函数等，更方便CSS的编写和维护。

sublime 快捷键总结

发表于 2016-05-23   |   分类于 技术   |     |   阅读次数

sublime 是我一直使用的前端编辑器，现将常用快捷键整理一下：

  • alt + shift + p: 打开命令面板（输入一些插件命令，最常用命令，没有之一）
  • Ctrl+P：搜索项目中的文件（模糊检索，实时检索结果，比eclipse 要快很多）
  • ctrl+r: 搜索当前文件中方法定义处，文件方法很多时，可以快速找到方法定义
  • alt+shift+数字：编辑器分屏功能，数字是几就分成几个屏幕，工作中喜欢分成两个屏幕，一半 html 文件、一半js文件，方便两边 copy一些属性定义，查错等。
